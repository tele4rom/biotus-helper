# 📋 Зміни Формату Відповіді для Карточок Товарів

## ✅ Що Змінилося

Чат-бот тепер повертає товари у **структурованому JSON форматі** замість текстового опису.

## 🎯 Формат Відповіді

```json
{
  "response": "З радістю допоможу! Ось відмінні комплекси для імунітету:",
  "sessionId": "uuid",
  "productsFound": 3,
  "products": [
    {
      "id": "product-123",
      "title": "Вітаміни для імунітету, Immune Renew",
      "brand": "Now Foods",
      "price": "653 UAH",
      "article": "NOW-12345",
      "image": "https://vitahub.com.ua/image/...",
      "link": "https://vitahub.com.ua/travy-ta-gryby/...",
      "reason": "Цей комплекс підтримує захисні сили організму"
    }
  ]
}
```

## 🎨 Дизайн Карточки

```
┌─────────────────────────────────────┐

│                                     │

│                                     │

```
